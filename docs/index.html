🧜 mermaid.nvim

Preview, format, lint, and render Mermaid diagrams directly inside Neovim —
with zero external dependencies for the core features.

📦 Install from GitHub ↓ Features

🚀 Features

🖥️ Live Preview

Real-time browser preview via built-in Lua HTTP server with SSE. Updates as you type.

No deps

🎨 Dual Renderers

Standard mermaid.js or aesthetic beautiful-mermaid — switch with one config option.

No deps

🔲 Floating Panel

In-editor floating window showing preview URL, connection status, and quick actions.

No deps

📟 Inline Render

Render diagrams directly in Kitty or chafa-capable terminals.

mmdc + chafa

✏️ Auto-Format

Smart indentation and token spacing for all Mermaid diagram types.

No deps

🔍 Diagnostics

Error checking via vim.diagnostic with mermaid-cli integration.

mmdc

🌓 Dark Mode

Auto-detects Neovim's background setting and syncs the preview live.

No deps

⚡ Zero Config

Works out of the box. Set up with 4 lines of Lua.

No deps

📋 Commands

CommandDescription
:MermaidPreviewOpen live browser preview
:MermaidPreviewStopStop the preview server
:MermaidFormatAuto-format current buffer
:MermaidRenderRender inline in terminal
:MermaidCopyURLCopy preview URL to clipboard

⚙️ Quick Start

Install via lazy.nvim:

return {
    "kevalin/mermaid.nvim",
    dependencies = { "nvim-treesitter/nvim-treesitter" },
    config = function()
        require("mermaid").setup()
    end,
}

Then open a .mmd file and run :MermaidPreview.
